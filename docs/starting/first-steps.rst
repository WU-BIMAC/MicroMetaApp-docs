The following tutorial will guide you through your first steps with Micro-Meta App, and introduce you to the main features, buttons and concepts you need to get started.
You will see how to:

* Launch Micro-Meta App
* Create a Microscope file
* Edit basic Microscope file attributes
* Add hardware components to the Microscope file
* Edit hardware components attributes
* Save a Microscope file
